# CERCASP-OS

Overview: CERCASP-OS | RFC CRC2302227N7 | Fundado 22/02/2023

This is the monorepo for the CERCASP website.

## Apps

- `apps/web`: Next.js 15 Frontend
- `apps/api`: NestJS 12 Backend

## Packages

- `packages/ui`: Shadcn/Tailwind Components
- `packages/db`: Prisma Schemas/Migrations
- `packages/utils`: Helpers/Validators
- `packages/audit`: Blockchain Trazabilidad

## Getting Started

1. Install dependencies: `yarn install`
2. Start development: `yarn dev`
3. Build for production: `yarn build`
